# विंडोज़ वॉल्ट

विंडोज़ वॉल्ट, वॉल्ट उत्पाद परिवार का मूल विंडोज़ सदस्य है। यह एक .NET 8 WPF एप्लिकेशन है जिसमें WebView2 संपादक, मूल एप्लिकेशन इन्वेंट्री, प्रवर्तन इंजन, कस्टम-नियम रनटाइम और एक स्थानीय वेब-ऐप ब्रिज हब है।

कोड उत्पाद अनुबंध है. इन-ऐप मैनुअल बनाए रखा गया है [src/WindowsBlocker/WebAssets/manual/en.md](src/WindowsBlocker/WebAssets/manual/en.md)।

## वर्तमान क्षमताएँ

- चयनित विंडोज़ अनुप्रयोगों के लिए डिफ़ॉल्ट समूह और उन्नत नीति नियमों के लिए कस्टम समूह।
- तत्काल, भत्ता और उलटी गिनती मोड; अनुसूचियाँ; जमाना; झपकी लेना; और समूह आयात/निर्यात।
- विंडोज़ एप्लिकेशन इन्वेंट्री और विंडो-आधारित प्रवर्तन घटक।
- `src/WindowsBlocker/WebAssets/` से होस्ट किया गया एक WebView2 संपादक।
- सिंटैक्स जांच और लॉग फ़ीड के साथ नियंत्रित कस्टम-नियम निष्पादन।
- स्पष्ट रूप से जुड़े संगत समूहों के लिए एक लूपबैक ब्रिज हब।
- नेटिव टाइमर, टोस्ट और पैनल ओवरले विंडो।

## निर्माण

चेक-इन समाधान और प्रोजेक्ट का उपयोग करें:

```powershell
dotnet build WindowsBlocker.sln
```

एप्लिकेशन प्रोजेक्ट `net8.0-windows` को लक्षित करता है और WPF प्लस WebView2 का उपयोग करता है। आवश्यक .NET SDK और WebView2 रनटाइम उपलब्ध होने पर इसे विंडोज़ पर बनाएं और चलाएं।

## प्रोजेक्ट मानचित्र

| क्षेत्र | स्रोत निर्देशिका |

| समूह मॉडल और नीति मूल्यांकन | `src/WindowsBlocker/Core/` |
| मूल प्रवर्तन | `src/WindowsBlocker/Enforcement/` |
| ऐप इन्वेंट्री और वेबव्यू ब्रिज | `src/WindowsBlocker/WebUI/` |

## दस्तावेज़ीकरण और अनुवाद

अंग्रेजी दस्तावेज़ विहित रहते हैं। यूआई लेबल `src/WindowsBlocker/WebAssets/translation/` में संपूर्ण JSON कैटलॉग का उपयोग करते हैं; अनुवादित मैनुअल `manual/en.md` के पास रहते हैं, और शेष बनाए गए दस्तावेजों की अनुवादित प्रतियां `i18n-docs/<locale>/` के अंतर्गत हैं।
