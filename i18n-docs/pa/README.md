# ਵਿੰਡੋਜ਼ ਵਾਲਟ

ਵਿੰਡੋਜ਼ ਵਾਲਟ ਵਾਲਟ ਉਤਪਾਦ ਪਰਿਵਾਰ ਦਾ ਮੂਲ ਵਿੰਡੋਜ਼ ਮੈਂਬਰ ਹੈ। ਇਹ ਇੱਕ .NET 8 WPF ਐਪਲੀਕੇਸ਼ਨ ਹੈ ਜਿਸ ਵਿੱਚ WebView2 ਸੰਪਾਦਕ, ਨੇਟਿਵ ਐਪਲੀਕੇਸ਼ਨ ਇਨਵੈਂਟਰੀ, ਇਨਫੋਰਸਮੈਂਟ ਇੰਜਣ, ਕਸਟਮ-ਰੂਲ ਰਨਟਾਈਮ, ਅਤੇ ਇੱਕ ਸਥਾਨਕ ਵੈਬ-ਐਪ ਬ੍ਰਿਜ ਹੱਬ ਹੈ।

ਕੋਡ ਉਤਪਾਦ ਦਾ ਇਕਰਾਰਨਾਮਾ ਹੈ। ਇਨ-ਐਪ ਮੈਨੂਅਲ ਹੈ [src/WindowsBlocker/WebAssets/manual/en.md](src/WindowsBlocker/WebAssets/manual/en.md)।

## ਮੌਜੂਦਾ ਸਮਰੱਥਾਵਾਂ

- ਚੁਣੀਆਂ ਵਿੰਡੋਜ਼ ਐਪਲੀਕੇਸ਼ਨਾਂ ਲਈ ਡਿਫੌਲਟ ਸਮੂਹ ਅਤੇ ਉੱਨਤ ਨੀਤੀ ਨਿਯਮਾਂ ਲਈ ਕਸਟਮ ਸਮੂਹ।
- ਤੁਰੰਤ, ਭੱਤਾ, ਅਤੇ ਕਾਉਂਟਡਾਊਨ ਮੋਡ; ਸਮਾਂ-ਸਾਰਣੀ; ਫ੍ਰੀਜ਼; ਸਨੂਜ਼; ਅਤੇ ਸਮੂਹ ਆਯਾਤ/ਨਿਰਯਾਤ।
- ਵਿੰਡੋਜ਼ ਐਪਲੀਕੇਸ਼ਨ ਇਨਵੈਂਟਰੀ ਅਤੇ ਵਿੰਡੋ-ਅਧਾਰਤ ਲਾਗੂ ਕਰਨ ਵਾਲੇ ਹਿੱਸੇ।
- `src/WindowsBlocker/WebAssets/` ਤੋਂ ਹੋਸਟ ਕੀਤਾ ਗਿਆ ਇੱਕ WebView2 ਸੰਪਾਦਕ।
- ਸੰਟੈਕਸ ਜਾਂਚ ਅਤੇ ਲੌਗ ਫੀਡ ਦੇ ਨਾਲ ਨਿਯੰਤਰਿਤ ਕਸਟਮ-ਨਿਯਮ ਐਗਜ਼ੀਕਿਊਸ਼ਨ।
- ਸਪੱਸ਼ਟ ਤੌਰ 'ਤੇ ਲਿੰਕ ਕੀਤੇ ਅਨੁਕੂਲ ਸਮੂਹਾਂ ਲਈ ਇੱਕ ਲੂਪਬੈਕ ਬ੍ਰਿਜ ਹੱਬ।
- ਨੇਟਿਵ ਟਾਈਮਰ, ਟੋਸਟ, ਅਤੇ ਪੈਨਲ ਓਵਰਲੇ ਵਿੰਡੋਜ਼।

## ਬਣਾਓ

ਚੈੱਕ-ਇਨ ਹੱਲ ਅਤੇ ਪ੍ਰੋਜੈਕਟ ਦੀ ਵਰਤੋਂ ਕਰੋ:

```powershell
dotnet build WindowsBlocker.sln
```

ਐਪਲੀਕੇਸ਼ਨ ਪ੍ਰੋਜੈਕਟ `net8.0-windows` ਨੂੰ ਨਿਸ਼ਾਨਾ ਬਣਾਉਂਦਾ ਹੈ ਅਤੇ WPF ਪਲੱਸ WebView2 ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ। ਲੋੜੀਂਦੇ .NET SDK ਅਤੇ WebView2 ਰਨਟਾਈਮ ਉਪਲਬਧ ਹੋਣ ਦੇ ਨਾਲ ਇਸਨੂੰ ਵਿੰਡੋਜ਼ 'ਤੇ ਬਣਾਓ ਅਤੇ ਚਲਾਓ।

## ਦਸਤਾਵੇਜ਼ ਅਤੇ ਅਨੁਵਾਦ

ਅੰਗਰੇਜ਼ੀ ਦਸਤਾਵੇਜ਼ ਕੈਨੋਨੀਕਲ ਰਹਿੰਦੇ ਹਨ। UI ਲੇਬਲ `src/WindowsBlocker/WebAssets/translation/` ਵਿੱਚ ਪੂਰੇ JSON ਕੈਟਾਲਾਗ ਦੀ ਵਰਤੋਂ ਕਰਦੇ ਹਨ; ਅਨੁਵਾਦਿਤ ਮੈਨੂਅਲ `manual/en.md` ਦੇ ਕੋਲ ਰਹਿੰਦੇ ਹਨ, ਅਤੇ ਬਾਕੀ ਰੱਖੇ ਦਸਤਾਵੇਜ਼ਾਂ ਦੀਆਂ ਅਨੁਵਾਦਿਤ ਕਾਪੀਆਂ `i18n-docs/<locale>/` ਦੇ ਹੇਠਾਂ ਹਨ।
