#วินโดวส์ห้องนิรภัย

Windows Vault เป็นสมาชิก Windows ดั้งเดิมของตระกูลผลิตภัณฑ์ Vault เป็นแอปพลิเคชัน .NET 8 WPF ที่มีตัวแก้ไข WebView2, รายการแอปพลิเคชันแบบเนทีฟ, เอ็นจิ้นการบังคับใช้, รันไทม์กฎที่กำหนดเอง และฮับบริดจ์เว็บแอปในเครื่อง

รหัสคือสัญญาผลิตภัณฑ์ คู่มือในแอปที่ได้รับการปรับปรุงคือ [src/WindowsBlocker/WebAssets/manual/en.md](src/WindowsBlocker/WebAssets/manual/en.md)

## ความสามารถในปัจจุบัน

- กลุ่มเริ่มต้นสำหรับแอปพลิเคชัน Windows ที่เลือกและกลุ่มแบบกำหนดเองสำหรับกฎนโยบายขั้นสูง
- โหมดทันที ค่าเผื่อ และนับถอยหลัง ตารางเวลา; แช่แข็ง; งีบหลับ; และกลุ่มนำเข้า/ส่งออก
- รายการแอปพลิเคชัน Windows และส่วนประกอบการบังคับใช้ตามหน้าต่าง
- โปรแกรมแก้ไข WebView2 โฮสต์จาก `src/WindowsBlocker/WebAssets/`
- ควบคุมการดำเนินการกฎแบบกำหนดเองพร้อมการตรวจสอบไวยากรณ์และฟีดบันทึก
- ฮับบริดจ์แบบย้อนกลับสำหรับกลุ่มที่เข้ากันได้ที่เชื่อมโยงอย่างชัดเจน
- ตัวจับเวลาแบบเนทีฟ ขนมปังปิ้ง และหน้าต่างซ้อนทับแผง

## สร้าง

ใช้โซลูชันที่เช็คอินและโปรเจ็กต์:

```powershell
dotnet build WindowsBlocker.sln
```

โครงการแอปพลิเคชันกำหนดเป้าหมาย `net8.0-windows` และใช้ WPF บวกกับ WebView2 สร้างและรันบน Windows ด้วยรันไทม์ .NET SDK และ WebView2 ที่จำเป็น

## แผนที่โครงการ

| พื้นที่ | ไดเรกทอรีต้นทาง |
| --- | --- |
| แบบจำลองกลุ่มและการประเมินนโยบาย | `src/WindowsBlocker/Core/` |
| การบังคับใช้พื้นเมือง | `src/WindowsBlocker/Enforcement/` |
| พื้นที่โฆษณาของแอปและบริดจ์ WebView | `src/WindowsBlocker/WebUI/` |
| รันไทม์กฎที่กำหนดเอง | `src/WindowsBlocker/Rules/` |
| สะพานฮับ | `src/WindowsBlocker/Bridge/` |
| หน้าต่าง WPF และภาพซ้อนทับ | `src/WindowsBlocker/` |

## เอกสารและคำแปล

เอกสารภาษาอังกฤษยังคงเป็นมาตรฐาน ป้ายกำกับ UI ใช้แคตตาล็อก JSON ที่สมบูรณ์ใน `src/WindowsBlocker/WebAssets/translation/`; คู่มือที่แปลจะอยู่ข้างๆ `manual/en.md` และสำเนาที่แปลของเอกสารที่เก็บรักษาที่เหลือจะอยู่ภายใต้ `i18n-docs/<locale>/`
