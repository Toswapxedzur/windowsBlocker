# قبو ويندوز

Windows Vault هو عضو Windows الأصلي في عائلة منتجات Vault. إنه تطبيق .NET 8 WPF مزود بمحرر WebView2 ومخزون التطبيقات الأصلي ومحرك التنفيذ ووقت تشغيل القاعدة المخصصة ومركز جسر تطبيق الويب المحلي.

الرمز هو عقد المنتج. الدليل الموجود داخل التطبيق هو [src/WindowsBlocker/WebAssets/manual/en.md](src/WindowsBlocker/WebAssets/manual/en.md).

## الإمكانيات الحالية

- المجموعات الافتراضية لتطبيقات Windows المحددة والمجموعات المخصصة لقواعد السياسة المتقدمة.
- أوضاع فورية، وبدل، والعد التنازلي؛ الجداول الزمنية؛ تجميد؛ غفوة؛ واستيراد/تصدير المجموعة.
- مخزون تطبيقات Windows ومكونات الإنفاذ المستندة إلى النوافذ.
- محرر WebView2 مستضاف من `src/WindowsBlocker/WebAssets/`.
- التحكم في تنفيذ القواعد المخصصة من خلال التحقق من بناء الجملة وتغذية السجل.
- مركز جسر استرجاع للمجموعات المتوافقة المرتبطة بشكل صريح.
- الموقت الأصلي، والخبز المحمص، ونوافذ تراكب اللوحة.

## بناء

استخدم الحل والمشروع اللذين تم تسجيل الوصول إليهما:

```powershell
dotnet build WindowsBlocker.sln
```

يستهدف مشروع التطبيق `net8.0-windows` ويستخدم WPF plus WebView2. قم بإنشائه وتشغيله على نظام التشغيل Windows باستخدام وقت تشغيل .NET SDK وWebView2 المطلوب.

##التوثيق والترجمات

تظل الوثائق الإنجليزية أساسية. تستخدم تسميات واجهة المستخدم كتالوجات JSON الكاملة في `src/WindowsBlocker/WebAssets/translation/`؛ توجد الأدلة المترجمة بجانب `manual/en.md`، والنسخ المترجمة من المستندات المتبقية الموجودة تحت `i18n-docs/<locale>/`.
