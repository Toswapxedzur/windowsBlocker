# উইন্ডোজ ভল্ট

Windows Vault হল ভল্ট পণ্য পরিবারের নেটিভ উইন্ডোজ সদস্য। এটি একটি WebView2 সম্পাদক, নেটিভ অ্যাপ্লিকেশন ইনভেন্টরি, এনফোর্সমেন্ট ইঞ্জিন, কাস্টম-রুল রানটাইম এবং একটি স্থানীয় ওয়েব-অ্যাপ ব্রিজ হাব সহ একটি .NET 8 WPF অ্যাপ্লিকেশন।

কোড হল পণ্য চুক্তি। অ্যাপ-মধ্যস্থ ম্যানুয়ালটি হল [src/WindowsBlocker/WebAssets/manual/en.md](src/WindowsBlocker/WebAssets/manual/en.md)।

## বর্তমান ক্ষমতা

- নির্বাচিত উইন্ডোজ অ্যাপ্লিকেশনের জন্য ডিফল্ট গ্রুপ এবং উন্নত নীতির নিয়মের জন্য কাস্টম গ্রুপ।
- অবিলম্বে, ভাতা, এবং গণনা মোড; সময়সূচী; জমাট স্নুজ এবং গ্রুপ আমদানি/রপ্তানি।
- উইন্ডোজ অ্যাপ্লিকেশন ইনভেন্টরি এবং উইন্ডো-ভিত্তিক প্রয়োগকারী উপাদান।
- `src/WindowsBlocker/WebAssets/` থেকে হোস্ট করা একটি WebView2 সম্পাদক।
- সিনট্যাক্স চেকিং এবং একটি লগ ফিড সহ নিয়ন্ত্রিত কাস্টম-রুল এক্সিকিউশন।
- সুস্পষ্টভাবে লিঙ্কযুক্ত সামঞ্জস্যপূর্ণ গোষ্ঠীগুলির জন্য একটি লুপব্যাক ব্রিজ হাব৷
- নেটিভ টাইমার, টোস্ট এবং প্যানেল ওভারলে উইন্ডো।

## নির্মাণ করুন

চেক-ইন সমাধান এবং প্রকল্প ব্যবহার করুন:

```powershell
dotnet build WindowsBlocker.sln
```

অ্যাপ্লিকেশন প্রকল্পটি `net8.0-windows` লক্ষ্য করে এবং WPF প্লাস WebView2 ব্যবহার করে। প্রয়োজনীয় .NET SDK এবং WebView2 রানটাইম উপলব্ধ সহ উইন্ডোজে এটি তৈরি করুন এবং চালান৷
