# Хранилище Windows

Windows Vault — это родной член семейства продуктов Vault для Windows. Это приложение .NET 8 WPF с редактором WebView2, собственным инвентарем приложений, механизмом обеспечения соблюдения, средой выполнения настраиваемых правил и локальным мостовым концентратором веб-приложений.

Код — это контракт на продукт. Поддерживаемое руководство в приложении — [src/WindowsBlocker/WebAssets/manual/en.md](src/WindowsBlocker/WebAssets/manual/en.md).

## Текущие возможности

— Группы по умолчанию для выбранных приложений Windows и пользовательские группы для расширенных правил политики.
- Режимы немедленного, разрешенного и обратного отсчета; графики; заморозить; вздремнуть; и групповой импорт/экспорт.
- Инвентаризация приложений Windows и компоненты принудительного применения на основе окон.
- Редактор WebView2, размещенный на `src/WindowsBlocker/WebAssets/`.
- Контролируемое выполнение пользовательских правил с проверкой синтаксиса и записью журнала.
- Концентратор моста обратной связи для явно связанных совместимых групп.
- Собственные окна таймера, всплывающего уведомления и наложения панелей.

## Сборка

Используйте зарегистрированное решение и проект:

```powershell
dotnet build WindowsBlocker.sln
```

Проект приложения ориентирован на `net8.0-windows` и использует WPF плюс WebView2. Создайте и запустите его в Windows с необходимым .NET SDK и доступной средой выполнения WebView2.
